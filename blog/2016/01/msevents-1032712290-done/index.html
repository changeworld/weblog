2016/1/28 に開催された『ノウハウお伝えします! Team Foundation Server 最新版でのワークアイテム管理』で講演しました。

ノウハウお伝えします! Team Foundation Server 最新版でのワークアイテム管理

講演は 14:30-16:30 で、『従来のウォーターフォールのやり方にアジャイルのやり方を適用しようとしたら猛反対されたけど、適用するためにやったこと』というタイトルそのままの内容です。

今回は主催者側から資料を配布したいという要望があり、それに対応したところ

  • 配布資料: 43ページ
  • スライド: 100ページ超え

という結果になっています。配布資料よりスライドの方が多いなんて不思議なこともありますね？

Slideshare には配布資料版をアップしています。

『従来のウォーターフォールのやり方にアジャイルのやり方を適用しようとしたら猛反対されたけど、適用するためにやったこと』 from Takashi Takebayashi

講演中は随時参加者のみなさんからの質問に答えましたが、上司や部下、同僚…と事前に私が予想していたもの以外に猛反対する存在が“自分自身”という方もおられ、様々な立場の方が参加されていることがよく分かりました。また、講演後にも参加者の方からいくつかご質問をいただきましたが、今回私が伝えた内容が少しでもみなさんにとって何かしらの刺激となり、みなさんの現場カイゼンのお役に立てれば幸いです。
